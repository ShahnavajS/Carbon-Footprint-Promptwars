# EcoScore - Carbon Footprint Tracker & AI Sustainability Twin

EcoScore is a modern web application designed to help users track, simulate, and reduce their carbon footprint. By leveraging gamification, community challenges, predictive analytics, and Gemini AI-powered coaching, EcoScore empowers individuals to make sustainable lifestyle choices.

---

## 🌟 Key Features

1. **AI Sustainability Twin**: Get a personalized, dynamically evolving sustainability profile generated by Gemini, providing insights into your strengths and target areas for improvement.
2. **Future Impact Forecast**: Visualize a predictive trajectory of your carbon footprint and EcoScore (30, 90, and 180 days) under different effort scenarios.
3. **Life Simulator**: Compare lifestyle changes side-by-side (e.g. going vegetarian, taking the metro) to see their immediate impact on carbon savings, cost, and eco points before making real-world commitments.
4. **Voice Coach**: Interact with a friendly, conversational sustainability coach via voice commands powered by Gemini and browser speech synthesis.
5. **Interactive Data Visualizations**: Understand your environmental impact with beautiful category breakdowns, activity heatmaps, and tangible equivalent metrics (e.g. flights avoided, trees planted).
6. **Community & Gamification**: Complete daily and weekly sustainability challenges, level up your EcoScore, unlock badges, and share your achievements via dynamic cards.
7. **Demo Mode**: Showcase all high-level features instantly with a fully pre-seeded mock user and guided walkthroughs.

## 🛠️ Technology Stack

- **Frontend Framework**: [Next.js](https://nextjs.org/) (React 19, TypeScript)
- **Styling**: Tailwind CSS & Vanilla CSS
- **State Management**: Zustand
- **Database & Authentication**: Firebase Firestore, Firebase Authentication, and Firebase Cloud Functions
- **Backend & AI Service**: Firebase Admin SDK & Google Gemini AI API
- **Testing Suite**: Vitest (Unit & Integration) & Playwright (E2E)

## 🚀 Getting Started

### Prerequisites

- Node.js (version 20 or higher)
- npm or yarn
- Firebase CLI (for local emulation)

### Setup & Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/ShahnavajS/Carbon-Footprint-Promptwars.git
   cd Carbon-Footprint-Promptwars
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Configure Environment Variables:**
   Copy `.env.example` to `.env.local` and configure your credentials:
   ```bash
   cp .env.example .env.local
   ```

4. **Run Firebase Emulators:**
   Start local Authentication and Firestore emulators:
   ```bash
   npm run emulators
   # or on Windows (PowerShell):
   npm run emulators:win
   ```

5. **Start Next.js Development Server:**
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) to view the application.
